# Purescript application scaffold

## Features

* Makefile-based compilation and running
* Generates static websites
* Includes React

## Requirements

* Haskell & Cabal
* [Purescript](http://www.purescript.org): `cabal install purescript`
* NPM
* [Bower](http://bower.io): `npm install -g bower`

## Building

`make all`

## Running

`make run`
